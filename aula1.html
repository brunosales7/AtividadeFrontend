Aula 1 - Git

Comandos poderosos para gerenciar o versionamento de sistemas.


Descrição Comando
Ver a versão git --version
Criar a pasta mkdir nome da pasta
Inicializar informações git status
Listar recurso da pasta ls
Navegar nas pastas cd ou cd ..
Abrir nano nano nome do arquivo
Fechar nano CTRL + X
Quem é você git config --local user.name"seu nome" e useremail"seu email"
Listar os commits git log ou git log --oneline ou git log -p obs:use q para sair
Ignorar arquivo Criar o arquivo .gitignore ,depois adicionar e "commitar"
Listar repositório remoto git remote
Adicionar repositório git remote add origin url do repositório
Enviar para o remoto com -u git push -u origin master,depois use git push
Desfazer um commit git revert coloque o hash do seu commit
